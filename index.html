// Recent Project

Nami

Nami is an open-source serverless framework that helps deploy an event driven system that handles bursty webhook traffic by setting up an API gateway with a FAAS platform and a message broker with eventual storage of data in a data store.

Nami is built using Node.js and AWS SDK. Cloud components are custom configured and deployed programmatically using AWS API's.
