Hello, I'am Sachin Chandy and I'am a fullstack software engineer based in London, UK.

I'am the co-creater of Nami, a serverless framework for consuming webhooks at scale.

The Nami framework helps deploy a system that can handle bursty webhook traffic by setting up an API endpoint with a FAAS platform and a message broker with eventual storage of data in a data store.

I strive to apply scientific thought to problem solving, focus on building strong fundamentals and consider myself language and framework agnostic.

I have experience with messaging queues, container-based deployments, event driven and microservices architecture. I'm fluent with Ruby and JavaScript, and am proficient with SQL, React, NodeJS, Ruby on Rails, Docker, AWS Lambda's, SQS queues and MongoDB.
